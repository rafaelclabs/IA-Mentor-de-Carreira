# IA-Mentor-de-Carreira
IA Mentor de Carreira é uma aplicação que utiliza inteligência artificial para orientar usuários na escolha de carreiras na área de tecnologia. Através da análise de interesses e objetivos profissionais, o sistema sugere áreas de atuação, habilidades necessárias e possíveis trilhas de aprendizado, auxiliando no desenvolvimento profissional.
